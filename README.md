# Bikeshare Project

## Description
The Python-script processes data from CSV files and provides various statistical insights into bikesharing usage.

## Requirements
This project requires **Python 3** and the following libraries installed:
* [pandas](https://pandas.pydata.org/)
* [numpy](https://numpy.org/)

## Files used
* bikeshare_starter.py
* Data source file (see Prerequisites)

## Prerequisites
To run this script, you must provide the source CSV files (Chicago, New York City, Washington). Place the files in the same folder as the Python script.
